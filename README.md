# AI Interview System — Intelligent Skill Assessment

An AI-powered interview platform built with Streamlit and Groq AI.

## Features
- AI question generation based on domain and difficulty
- Intelligent answer evaluation with skill scoring
- Auto difficulty adjustment based on performance
- Resume upload and personalized interview
- PDF report download
- Leaderboard
- Proctoring system

## Tech Stack
Python · Streamlit · Groq AI · SQLite · Plotly · ReportLab

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Groq API key to `.env`: `GROQ_API_KEY=your_key`
4. Run: `streamlit run app.py`
